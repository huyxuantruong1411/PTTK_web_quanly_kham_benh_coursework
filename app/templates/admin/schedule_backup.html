Cấu hình Backup tự động

Cấu hình lịch trình backup
Hủy
Hướng dẫn thiết lập backup tự động
Lưu ý: Để sử dụng tính năng backup tự động, bạn cần thiết lập cron job (trên Linux) hoặc Task Scheduler (trên Windows) để chạy script sau vào thời gian đã cấu hình.
Trên Linux (cron job):
0 2 * * * cd /path/to/your/project && python -c "from app.tasks import auto_backup; auto_backup()" > /dev/null 2>&1
Trên Windows (Task Scheduler):
  1. Mở Task Scheduler
  2. Tạo Basic Task mới
  3. Đặt tên và mô tả cho task
  4. Chọn tần suất chạy (Daily, Weekly, Monthly)
  5. Đặt thời gian chạy
  6. Chọn "Start a program"
  7. Đường dẫn program: python
  8. Arguments: -c "from app.tasks import auto_backup; auto_backup()"
  9. Start in: /path/to/your/project
